# Alaska GSoC Figures and Assets

This directory contains visual assets and figures used in the Alaska GSoC documentation.

## Current Assets

- `Alaska.png` - Official Alaska GSoC logo used in the main README (150x150px)

## Usage Guidelines

When using these assets:
- Maintain the original aspect ratio of the Alaska logo
- The logo should be used at 150x150px size for consistency with the main README
- Ensure proper attribution to the University of Alaska and Alaska Developer Alliance

## Contributing New Assets

If you need to add new figures or visual assets:
1. Use standard web formats (PNG, JPG, SVG)
2. Optimize images for web use (reasonable file sizes)
3. Include descriptive filenames
4. Update this README with new asset descriptions
5. Consider accessibility - provide alt text descriptions when used in documentation

## Asset Sources

- Alaska logo: Created for the Alaska GSoC initiative representing the 49th state and its open-source community
